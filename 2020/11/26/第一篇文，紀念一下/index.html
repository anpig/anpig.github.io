第一篇文，紀念一下

弄完覺得我好笨，又在瘋狂搞烏龍了。
我一開始以為要從 root 的 _config.yml 下去改，
還想說怎麼可以改的東西那麼少。
現在想想真的很笨，每個 theme 的設計都不一樣，
當然是從 themes 裡面的 _config.yml 去改，
後來想發文發不出來，去看教學影片才知道有格式。
不過弄一弄覺得還蠻有成就感的，
不用學 HTML, CSS, JavaScript 也可以寫 Blog，
真是太舒服啦！
