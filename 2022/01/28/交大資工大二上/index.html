交大資工大二上

紀錄  |  Anpig  |  2022-01-28  |  1.6k  |  5 Minutes

修課

服務學習（二）

這學期跟上學期一樣選了自由軟體組，期初的說明會跟上學期一模一樣，於是我的（期末再交的）期中報告也交了跟上學期一模一樣的。

基礎程式設計

系上必修的程式檢定考，沒考過有擋修，每個學期會舉辦好幾次，有人數限制要抽籤而且學長姐優先，結果我一次都沒抽到，我也不想去候補就退掉了。

數位電路實驗

這學期最後悔修的課，我對硬體的邏輯和 Verilog 這個語言掌握得非常的糟，每周的 Lab 都寫得非常痛苦，常常不知道自己哪裡錯了、常常要熬夜，最後的期末專題找了大神隊友躺分（物理上的躺，最後真正在做的時候我發燒躺在床上）看了大神的 Code 還有那種「喔～原來可以這樣寫～」的感覺，不過因為時間分配爛了加上團隊溝通和協作能力不佳導致最後的結果其實不太好（但我其實也沒資格說什麼）。如果沒有要往硬體發展的人請不要來修這門課。

計算機系統管理

System Administration，簡稱 SA，我覺得它是這學期選到最有趣也最實用的課，修完之後對 UNIX 有比較熟，這門課評分的重點是作業：

  • 作業一：安裝 FreeBSD
  • 作業二：寫 Shell Script 來做一個簡單的 TUI 系統管理工具
  • 作業三：FTP & ZFS backup
  • 作業四：nginx（或 Apache）反向代理、MySQL、Web Socket
  • 作業五：NFS & Firewall

每次作業至少都要花掉整個周末，很吃查資料的能力，麻煩但有趣，組隊修的話應該會輕鬆很多。考試就大多都是考古題，可以帶無限多紙本資料去考試，考古印一印都可以80以上。

機率（英文授課）

老師很認真也很會教但我太笨了所以上了一學期都不知道自己在幹嘛，最後也不知道為什麼摸到 A-，比我預期的還高。

演算法概論（英文授課）

課程涵蓋了排序、樹、遞迴、主定理、動態規劃、圖論，因為是概論課所以不是很理論，難易適中，總共有 6 次 Lab，每次有 5 ~ 7 個程式題目，每次可以寫 2 ~ 3 週，寫完大概有 1 ~ 2 週休息。期中期末都有筆試跟上機考。原本以為至少可以拿 A ，結果我筆試拿太低，最後甚至是四捨五入才有 A-，我超難過，好遮很遮吧？

網路程式設計概論

開學前最期待的課，結果期中考後的課我幾乎都沒上。一開始先介紹傳輸層，比計網概教得再更深一點，再來就教我們怎麼用 C 的函式庫建立 TCP/UDP socket、怎麼做 multiplexing、address conversion等等的（我只上到這些，尷尬），總共有三次作業兩次上機考，每個都佔 20%：

  • 作業一：寫一個 BBS Server，可以讓使用者

    • 註冊
    • 登入
    • 登出
    • 看自己 ID
    • 退出
    • 收發信件
    • 至少可以同時讓一個 Client 使用

  • 作業二：寫一個 BBS Server，可以讓使用者

    • 註冊
    • 登入
    • 登出
    • 看自己 ID
    • 退出
    • 建立看板
    • 查看看板列表
    • 建立貼文
    • 查看貼文列表
    • 編輯貼文
    • 推文
    • 至少可以同時讓 10 個 Clients 使用

  • 作業三：寫一個 BBS Server，可以讓使用者

    • 註冊
    • 登入
    • 登出
    • 退出
    • 使用聊天室
      • 使用 UDP
      • 處理兩種不同特定的封包格式
      • 實作會自動 Ban 人的河蟹功能
    • 能同時讓多個 Clients 使用。

  • 期中上機考：

    • echo：寫一個 Client 把 Server 傳來的訊息傳回去。
    • 解迷宮：寫一個 Client 解 Server 傳來的迷宮，有不同難度，最簡單的地圖沒有任何牆壁，最難的地圖很大需要捲動。

  • 期末上機考：

    • echo：一些同時使用 TCP 和 UDP 的酷東西。
    • 踩地雷：自動玩，特殊封包格式。

上機考如果寫不完一週內可以回去繼續寫，回去寫的部分打折，我覺得這個作法很好，用分數鼓勵大家練習。這門課的作業寫起來很好玩，我很喜歡。但是作業三炸了，多人同時使用的情況沒有處理好，直接少了 40 分，甚至上傳格式沒弄好又少了 10 分，直接從 A+ 變 A-，我真的遮爛了。

會計學

三學分校基本涼通識，有作業但老師給的講義裡都有解答，照抄就好，這門課是英文授課，這學期採線上遠距教學，要點名，但其實也只是大家進入 Google Meet 看老師播自己上課影片而已，一開始我真的不懂為什麼還要特意這樣浪費彼此時間而不是發影片讓我們自己看（像大一物理那樣），所以我都點完名就高歌離席了，直到考完期末考我才恍然大悟這麼做是為了讓基本分數有個依據（老師說這門課有出席期中期末就會過了），我計算機按錯結果整個大題全錯，考了一個很遮的分數，學期成績也很遮，好遮吧？

韓文（一）

老師很注重發音，所以花了很多時間在這部分，而會話的部分就比較少。需要背蠻多單字，小考會考，需要花一些時間讀。有兩次作業，第一次是造句，第二次是短文自我介紹，當作期末口試要上台念給全班聽，除了介紹自己之外也可以介紹家人或喜歡的團體等等，我介紹了小吉帳號 @pooomama 裡的三隻小吉。老師很會跟學生聊天，雖然我們班感覺比較害羞回應比較少，但上課氣氛整體來說還是很不錯。

其他

  • 學期末組了一台桌機開始玩 PVE，蠻好玩的，等我哪天想寫再寫一篇。
  • 去面試系計中結果被刷掉，遮爛了。

總結

這學期的我是個不稱職的分奴，把分數看得很重卻表現和其中的我的期待有落差，不過比起大一已經進步很多了。

雖然比較忙又比較累，不過少了大一死人必修我過得很快樂。
